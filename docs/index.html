01Premise

Real vim, in a single static page.

Ripple is a full Vim editor in the browser. Modal editing — Insert, Normal, Visual, Replace — with motions, operators, text objects, registers, macros, and . repeat. Backed by @replit/codemirror-vim over CodeMirror 6.

Splits and tabs work as you'd expect from :vsplit, :tabnew, Ctrl-W h/j/k/l, gt/gT. A localStorage-backed virtual filesystem persists files across sessions; seven examples ship by default.

03Quickstart

Build, deploy anywhere static.

esbuild produces a single bundle. Output sits in dist/ — upload to GitHub Pages, GitLab Pages, S3, nginx, or open dist/index.html from a file:// URI.

# Prereqs: Node.js 18+
$ git clone https://github.com/Real-Fruit-Snacks/Ripple.git
$ cd Ripple
$ npm install
$ npm run build       # production (minified) → dist/
$ npm run dev         # development with watch + live server

# Verify
$ ls -la dist/
$ npx eslint src/

05Architecture

Vanilla DOM, single bundle.

Pure JavaScript with CodeMirror 6 for the editor core and @replit/codemirror-vim for Vim emulation. No framework — direct DOM APIs for performance and zero overhead.

src/
main.js              // Bootstrap · register commands
editor.js            // CodeMirror EditorState/EditorView factory
theme.js             // Catppuccin Mocha palette + highlights
languages.js         // Language detection and loading
commands.js          // Ex command definitions (:w, :e, :q, …)
keymaps.js           // Vim action mappings (gt, Ctrl-W, …)
fs/
  VirtualFS.js       // localStorage-backed filesystem
  defaults.js        // Default example files
ui/
  App.js             // Main shell and lifecycle
  TabBar.js          // Tab strip with modified indicators
  StatusLine.js      // Mode badge, position, file info
  Splits.js          // Binary tree split pane manager
  Splash.js          // Vim-style welcome screen

build.js             // esbuild script
dist/                // Build output (deploy this)
  index.html
  vim.js             // Single bundled JS (~740 KB)
