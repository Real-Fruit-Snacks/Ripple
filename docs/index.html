FEATURES

Full Vim, zero setup

Insert, Normal, Visual, and Replace modes with motions, operators, text objects, registers, macros, and dot repeat. All in one HTML file.

Full Vim Mode

Insert, Normal, Visual, and Replace modes with motions, operators, text objects, registers, macros, and dot repeat. Powered by @replit/codemirror-vim -- the most complete Vim implementation for the browser.

Movement: h j k l w b e 0 $ gg G { } %
Editing: i a I A o O r R x dd D yy p P
Visual: v V Ctrl-V
Search: / ? n N * #
740KB
Single Bundle

Split Panes

Horizontal and vertical splits with drag-to-resize dividers. Ctrl-W h/j/k/l spatial navigation. Multiple views of the same buffer with independent cursors.

Tab System

Tab bar with modified indicators, click-to-switch, and close buttons. gt/gT for tab cycling. Modified buffers protected from accidental close.

Virtual Filesystem

localStorage-backed file persistence across sessions. Seven example files ship by default. Standard :w/:e workflow with files surviving page refresh.

Syntax Highlighting

JavaScript, TypeScript, JSX, TSX, HTML, CSS, Python, Markdown, and JSON. Language detection by file extension. Powered by CodeMirror Lezer parsers.

ARCHITECTURE

Vanilla JS, no framework

Pure JavaScript with CodeMirror 6 for the editor core. esbuild produces a single minified bundle. Direct DOM APIs for maximum performance.

1

Editor Core

CodeMirror 6 EditorState/EditorView with @replit/codemirror-vim for Vim emulation. Lezer parser system for accurate, incremental syntax highlighting.

2

Split Manager

Binary tree split pane manager with drag-to-resize dividers. Spatial navigation via Ctrl-W keybindings. Multiple views of the same buffer with independent cursors.

3

Virtual Filesystem

localStorage-backed persistence with :w/:e/:rm commands. Seven default example files. File list via :files and buffer list via :ls.

4

Build Pipeline

esbuild script produces a single vim.js bundle (~740KB). Deploy dist/ to any static host. Inline critical CSS in the HTML shell for instant load.

GETTING STARTED

Build or deploy

Node.js 18+ to build. Output is a self-contained dist/ folder you can open directly or upload anywhere.

build.sh
$ git clone https://github.com/Real-Fruit-Snacks/Ripple.git
$ cd Ripple

$ npm install
$ npm run build
dist/index.html
dist/vim.js (740KB)

# Or development with live reload
$ npm run dev
commands.vim
# File operations
:e main.js       open file
:w               save buffer
:saveas new.js   save as

# Splits and tabs
:vsplit app.js   vertical split
:tabnew style.css new tab

# Batch
:wa  write all  :qa  quit all
