# Contributing to Ripple

## Development Setup
1. Clone the repository
2. Open `index.html` in a browser — no build step needed
3. Edit JavaScript files directly

## Code Style
- Vanilla JavaScript (no frameworks)
- ES2022+ features
- CodeMirror 6 API conventions

## Pull Request Process
1. Fork and create a feature branch
2. Test in Chrome, Firefox, and Safari
3. Submit PR with description of changes

## Commit Messages
Use conventional commits: `feat:`, `fix:`, `docs:`, `style:`, `refactor:`

No AI co-author signatures in commits.
